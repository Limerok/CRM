# CRM

Custom CRM admin panel inspired by the MVC structure of OcStore 2.3 / OpenCart.

## Требования
- PHP 5.6+ (рекомендуется 8.0)
- MySQL 5.7+

## Установка
1. Скопируйте файлы проекта на сервер.
2. Обновите при необходимости параметры подключения к БД в `config/config.php`.
3. Убедитесь, что сервер поддерживает расширение PDO для MySQL.
4. При первом запуске система автоматически создаст необходимые таблицы и администратора с логином/паролем `admin`.

## Структура
- `admin/` — административная панель (контроллеры, представления, стили).
- `catalog/` — заготовка для клиентской части.
- `system/` — ядро, миграции, классы БД.
- `config/` — конфигурация приложения.

## Возможности админки
- Авторизация с использованием сессий.
- Дашборд со сводной статистикой.
- Управление товарами, категориями и производителями.
- Управление поставками: создание поставки, обновление складских остатков, история поставок.
- Просмотр текущих остатков на складе.
- Оформление продаж с учетом остатка.

## UI
Панель построена на Bootstrap 5, включает адаптивную верстку и динамические подсказки при подборе товаров для поставок и продаж.
